\subsection{Detailed Description}
String in the (Java class file modified) UTF8 format

The string data can be both in the plain UTF8 format or in its Java class file variation, the string manager doesn't care about it, only the conversion functions do. The ones going from the Java class file modified format start with the jutf8\_\- prefix, the ones working on plain UTF8 strings begin with the utf8\_\- prefix. Note that the structure holds only a pointer to the next object in the hash table, and the data array is used as a C89-\/compatible variable size array.
