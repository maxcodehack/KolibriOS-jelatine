Converts the specified {\ttfamily \doxyref{String}{p.}{classjava_1_1lang_1_1_string}} into an {\ttfamily int} using the specified radix (base). The string must not be {\ttfamily null} or empty. It may begin with an optional '-\/', which will negate the answer, provided that there are also valid digits. Each digit is parsed as if by {\ttfamily Character.digit(d, radix)}, and must be in the range {\ttfamily 0} to {\ttfamily radix -\/ 1}. Finally, the result must be within {\ttfamily MIN\_\-VALUE} to {\ttfamily MAX\_\-VALUE}, inclusive. Unlike \doxyref{Double.parseDouble}{p.}{classjava_1_1lang_1_1_double_a5c5fabd1e39b3d9a8b2bc37277da15bc}, you may not have a leading '+'; and 'l' or 'L' as the last character is only valid in radices 22 or greater, where it is a digit and not a type indicator.
