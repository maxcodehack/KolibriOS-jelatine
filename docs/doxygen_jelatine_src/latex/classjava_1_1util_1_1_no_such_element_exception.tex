\subsection{Detailed Description}
Exception thrown when an attempt is made to access an element that does not exist. This exception is thrown by the \doxyref{Enumeration}{p.}{interfacejava_1_1util_1_1_enumeration}, Iterator and ListIterator classes if the nextElement, next or previous method goes beyond the end of the list of elements that are being accessed. It is also thrown by \doxyref{Vector}{p.}{classjava_1_1util_1_1_vector} and Stack when attempting to access the first or last element of an empty collection.
