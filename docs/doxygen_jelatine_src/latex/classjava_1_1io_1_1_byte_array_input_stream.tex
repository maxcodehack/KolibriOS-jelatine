This stream will read from position {\ttfamily offset} in the array for a length of {\ttfamily length} bytes past {\ttfamily offset}. If the stream is reset to a position before {\ttfamily offset} then more than {\ttfamily length} bytes can be read from the stream. The {\ttfamily length} value should be viewed as the array index one greater than the last position in the buffer to read.
