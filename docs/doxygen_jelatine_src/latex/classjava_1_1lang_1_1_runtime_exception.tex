\subsection{Detailed Description}
All exceptions which are subclasses of {\ttfamily \doxyref{RuntimeException}{p.}{classjava_1_1lang_1_1_runtime_exception}} can be thrown at any time during the execution of a Java virtual machine. Methods which throw these exceptions are not required to declare them in their throws clause.
