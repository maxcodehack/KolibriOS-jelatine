\subsection{Detailed Description}
Applications should not try to catch errors since they indicate abnormal conditions. An abnormal condition is something which should not occur, or which should not be recovered from. This latter category includes {\ttfamily ThreadDeath} and {\ttfamily AssertionError}.

A method is not required to declare any subclass of {\ttfamily \doxyref{Error}{p.}{classjava_1_1lang_1_1_error}} in its {\ttfamily throws} clause which might be thrown but not caught while executing the method.
