\subsection{Detailed Description}
This exception can be thrown to indicate an attempt to access an index which is out of bounds of a \doxyref{String}{p.}{classjava_1_1lang_1_1_string}. Any negative integer, and a positive integer greater than or equal to the size of the string, is an index which would be out of bounds.
