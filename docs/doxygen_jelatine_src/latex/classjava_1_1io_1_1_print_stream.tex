\subsection{Detailed Description}
This class prints Java primitive values and object to a stream as text. None of the methods in this class throw an exception. However, errors can be detected by calling the {\ttfamily \doxyref{checkError()}{p.}{classjava_1_1io_1_1_print_stream_a12ef6072467fa82489b5cf60e67b7c9a}} method. Additionally, this stream can be designated as \char`\"{}autoflush\char`\"{} when created so that any writes are automatically flushed to the underlying output sink when the current line is terminated. 

This class converts char's into byte's using the system default encoding.
