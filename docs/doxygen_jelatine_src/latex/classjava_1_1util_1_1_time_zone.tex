\subsection{Detailed Description}
This class represents a time zone offset and handles daylight savings.

You can get the default time zone with {\ttfamily getDefault}. This represents the time zone where program is running.

Another way to create a time zone is {\ttfamily getTimeZone}, where you can give an identifier as parameter. For instance, the identifier of the Central European Time zone is \char`\"{}CET\char`\"{}.

With the {\ttfamily getAvailableIDs} method, you can get all the supported time zone identifiers.
