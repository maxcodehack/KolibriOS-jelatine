\subsection{Detailed Description}
\doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} represents a single thread of execution in the VM. When an application VM starts up, it creates a non-\/daemon \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} which calls the main() method of a particular class. There may be other Threads running, such as the garbage collection thread.

Threads have names to identify them. These names are not necessarily unique. Every \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} has a priority, as well, which tells the VM which Threads should get more running time. New threads inherit the priority and daemon status of the parent thread, by default.

There are two methods of creating a \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread}: you may subclass \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} and implement the {\ttfamily \doxyref{run()}{p.}{classjava_1_1lang_1_1_thread_a1e00c525905d96211c184d515cee2418}} method, at which point you may start the \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} by calling its {\ttfamily \doxyref{start()}{p.}{classjava_1_1lang_1_1_thread_a9ab05f9deaab4d233d10c762ff1ab4be}} method, or you may implement {\ttfamily \doxyref{Runnable}{p.}{interfacejava_1_1lang_1_1_runnable}} in the class you want to use and then call new {\ttfamily Thread(your\_\-obj).\doxyref{start()}{p.}{classjava_1_1lang_1_1_thread_a9ab05f9deaab4d233d10c762ff1ab4be}}.

The virtual machine runs until all non-\/daemon threads have died (either by returning from the \doxyref{run()}{p.}{classjava_1_1lang_1_1_thread_a1e00c525905d96211c184d515cee2418} method as invoked by \doxyref{start()}{p.}{classjava_1_1lang_1_1_thread_a9ab05f9deaab4d233d10c762ff1ab4be}, or by throwing an uncaught exception); or until {\ttfamily \doxyref{System.exit}{p.}{classjava_1_1lang_1_1_system_a8a9bcfb622df80be0fef22f654d01fe8}} is called with adequate permissions.

It is unclear at what point a \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} should be added to a ThreadGroup, and at what point it should be removed. Should it be inserted when it starts, or when it is created? Should it be removed when it is suspended or interrupted? The only thing that is clear is that the \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} should be removed when it is stopped.
