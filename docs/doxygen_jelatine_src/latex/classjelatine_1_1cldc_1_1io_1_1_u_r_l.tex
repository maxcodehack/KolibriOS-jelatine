\subsection{Detailed Description}
This class stores \doxyref{URL}{p.}{classjelatine_1_1cldc_1_1io_1_1_u_r_l} format as described in RFC 2396. This takes the general form:\par
 \{scheme\}:[\{target\}][\{params\}]\par
 where \{scheme\} is the name of a protocol such as http.\par
 The \{target\} is normally some kind of network address.\par
 Any \{params\} are formed as a series of equates of the form \char`\"{};x=y\char`\"{}. Example: \char`\"{};type=a\char`\"{}.
