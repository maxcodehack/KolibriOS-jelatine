\subsection{Detailed Description}
Thrown when a thread interrupts another thread which was previously sleeping, waiting, or paused in some other way. See the {\ttfamily interrupt} method of class {\ttfamily \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread}}.
