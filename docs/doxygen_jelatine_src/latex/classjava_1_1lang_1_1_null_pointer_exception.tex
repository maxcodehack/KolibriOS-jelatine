\subsection{Detailed Description}
Thrown when attempting to use {\ttfamily null} where an object is required. The Virtual Machine automatically throws this exception for the following:\par

\begin{DoxyItemize}
\item Calling an instance method on a null object 
\item Accessing or modifying a field of a null object 
\item Taking the array length of a null array 
\item Accessing or modifying the slots of a null array 
\item Throwing a null \doxyref{Throwable}{p.}{classjava_1_1lang_1_1_throwable} 
\item Synchronizing on a null object 
\end{DoxyItemize}

Applications should also throw NullPointerExceptions whenever {\ttfamily null} is an inappropriate parameter to a method.
