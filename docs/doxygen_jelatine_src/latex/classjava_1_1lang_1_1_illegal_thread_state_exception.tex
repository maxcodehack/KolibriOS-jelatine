\subsection{Detailed Description}
Thrown When trying to manipulate a \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread} which is in an inappropriate state. Since the documentation suggests that this can happen with {\ttfamily Thread.suspend} or {\ttfamily Thread.resume}, but these two methods are deprecated, this exception is likely very rare.
