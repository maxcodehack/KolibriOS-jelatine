\subsection{Detailed Description}
Internal VM class used for finalizing other objects. This class uses a number of tricks to achieve its purpose. For example all objects which need to be finalized are cast to \doxyref{VMFinalizer}{p.}{classjelatine_1_1_v_m_finalizer} even though they do not inherit from the class.
