\subsection{Detailed Description}
This exception can be thrown to indicate an attempt to access an index which is out of bounds on objects like \doxyref{String}{p.}{classjava_1_1lang_1_1_string}, Array, or Vector. Usually any negative integer less than or equal to -\/1 and positive integer greater than or equal to the size of the object is an index which would be out of bounds.
