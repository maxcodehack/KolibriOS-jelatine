\subsection{Detailed Description}
\doxyref{Runnable}{p.}{interfacejava_1_1lang_1_1_runnable} is an interface you implement to indicate that your class can be executed as the main part of a \doxyref{Thread}{p.}{classjava_1_1lang_1_1_thread}, among other places. When you want an entry point to run a piece of code, implement this interface and override run.
